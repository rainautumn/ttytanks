# ttytanks 
The game is written in C for *nix system
```
                                       # #
                                       ###
                                        #                                      





   #
  ###
  # #
                                 #                                           # #
                                                                        #    ###
                                                                              #





  
                                        #
                                       ###
                                       # #
frags : 34
```

### Version
S 1.1

### run

```bash
$ git clone https://github.com/rainautumn/ttytanks.git
$ cd ttytanks
$ make static && ./static_ttytanks
```
### install ncurces lib
###### deb* linux
```
$ sudo apt install libncurses5-dev
```
###### FreeBSD
```
# pkg install ncurces
```
###### Gentoo
```
# emerge -av sys-libs/ncurses
```
###### Arch
```
# pacman -S ncurses
```
###### Fedora
```
# yum install ncurses-devel
```
